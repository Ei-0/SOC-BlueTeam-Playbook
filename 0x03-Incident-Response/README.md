## Overview
This section contains incident response playbooks designed to guide SOC analysts and responders during active security incidents.

Each playbook provides structured, repeatable actions to ensure consistent and effective response under pressure.

---

## Objectives
- Standardize incident response actions
- Minimize business and security impact
- Preserve evidence for investigation
- Support escalation and decision-making

---

## Playbook Structure
Each playbook includes:
- Purpose and scope
- Detection sources
- Initial triage actions
- Severity classification
- Containment, eradication, and recovery steps
- Escalation criteria
- Lessons learned

---

## Covered Incident Types
- Malware
- Phishing
- Insider Threat
- Ransomware

---

## Outcome
These playbooks demonstrate operational readiness and the ability to respond effectively to a wide range of security incidents.
